# Amazon E-Commerce Sales Analytics Dashboard (Power BI)

This repository contains an interactive Power BI dashboard created for analyzing Amazon E-Commerce product sales.  
It provides insights into sales performance, product trends, customer reviews, revenue patterns, and category-wise analysis.

## ğŸ“Œ Project Objective

To provide a comprehensive analytical view of Amazon product sales using Power BI by:

- Tracking total sales, orders, AOV, and customer reviews  
- Identifying top performing products and categories  
- Understanding time-based trends (monthly, weekly)  
- Analyzing discount impact and review patterns  
- Providing decision-making insights through interactive filters & visuals
